Weather App

Developing my Weather App using HTML, CSS, and Java was a deep dive into integrating live data into a functional interface. Understanding how to fetch and display real-time weather conditions required learning how APIs work and how Java can handle asynchronous data requests. Styling the UI with CSS taught me the importance of user-friendly design, while structuring the content with HTML reinforced the fundamentals of clean, accessible code. A key challenge was ensuring smooth transitions between weather states, which pushed me to experiment with dynamic styling and event-driven programming.
